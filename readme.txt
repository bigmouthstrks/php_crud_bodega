Éste módulo está conectado a la base de datos MySQL de nombre "sistemas_expertos", que cuenta con dos tablas: "Bodega" y "Producto".
Las funcionalidades del módulo son:
- Agregar, Modificar, Eliminar y Listar -> Productos
- Agregar, Modificar, Eliminar y Listar -> Bodegas

Las tecnologías utilizadas para este proyecto son:
- HTML
- Bootstrap 4
- PHP 7.4.12
- Javascript
- MySQL
